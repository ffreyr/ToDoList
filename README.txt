-Our goal on this project is create a simple tool to manage tasks and duties.

-You can:
    -Add your tasks.
    -Edit and delete your tasks.
    -Filter your tasks by completion status.
    -Tick completed tasks.

-How to use?
    -Write your task to the textbox, click on the add button.
    -If you complete that task click, on the checkbox.
    -Click "Completed" button for filtering completed tasks.
    -Click "Uncompleted" button for filtering uncompleted tasks.
    -Click "All" button for show all tasks.
    -Click "Delete" button for deleting the chosen task.
    -Click "Edit" button for editing the chosen task.

-We used HTML for basic structre of the web page.
 CSS for designing the web page.
 JavaScript for add interactivity to the web page.
